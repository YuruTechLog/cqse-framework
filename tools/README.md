# Cokes Tools Directory Rules

## ■ このディレクトリの役割（spec）

このディレクトリは
「Cokesの実行負荷を下げるための再利用可能な機能部品（tool）」を置く場所。

思想・知識・ログはここに置かない。

---

## ■ 追加してよいもの（spec条件）

以下すべてを満たすもののみ追加する：

- 複数のログ処理で再利用される
- 判断ではなく手順として定義できる
- 人間の操作回数を減らす
- 入力→出力が明確である

---

## ■ 追加してはいけないもの

- 単発のログ専用知識
- 思考・考察・意思決定
- distillation / knowledge に属する内容
- 記事・文章生成そのもの

---

## ■ 追加前チェック（gate）

追加前に必ず確認する：

- [ ] これは3回以上使うか？
- [ ] 入力と出力が明確か？
- [ ] 判断ではなく処理か？
- [ ] knowledgeではなくtoolか？
- [ ] distillationの一部ではないか？

1つでもNOなら追加しない

---

## ■ 原則（重要）

toolsは「増やす場所」ではなく
**“作業ステップを固定化して再利用するための機能レイヤー”である**

---

## ■ 補足（設計思想）

このディレクトリは
Cokesの中で「人間の思考回数を減らす領域」である
