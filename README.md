# CQSE: Cognitive-Quality Systems Engineering
> 認知・品質システム工学：AIと人間のインターフェースおよび工程設計のためのフレームワーク

CQSE（読み方: コークス / `koʊks`）は、AI（LLM）を実務工程に組み込む際に生じる **「人間側の認知負荷」と「手戻りコスト」を最小化し、品質が担保された安全な Handoff（手離れ）を達成するため** の工程設計・運用・再検証の統合フレームワークです。

---

と偉そうに書いているが、

製品の「品質維持」のために「人間が楽する」にはどうするか

を考える工学。

「人間が楽する」の一例に「認知負荷の軽減」を大々的に推しているが、上記の目的が達成できるのであれば、そこ（認知負荷の軽減という言葉）にこだわるつもりはない。

楽して、いいモノ作りたい。

工学と名乗るので、それを再現する方法（再現性）については整理していく。

読み方の「コークス」は、化石燃料のコークス（cokes）を意識して読みを決めた。
認知負荷の過剰摂取で燃え尽きる（バーンアウトする）人間が少しでも減るように、という燃料（リソース）への願いを込めている。

みんなで楽できる仕掛けを整理していけたらいいなぁ。
