# 比喩カタログ（Metaphor Catalog）

Approach Axisの概念を読者に伝えるための説明補助ツール集。
**これは定義ではなく伝達手段。** 概念定義は `glossary.md` を参照。

用途: 本文執筆時に「どの比喩で説明するか」を選ぶための参照。

## glossary.mdから移動した比喩

### 散弾銃→狙い撃ち
導入比喩。軸なし指示=散弾銃（偶然ヒット）/ AXIS=狙い撃ち（距離操作でヒット率向上）。
使いどころ: 第1章・導入部。AXISの価値を直感的に伝える。

### 媒質三分割（Medium Trifurcation）
AI内部の説明モデル。AI=水槽 / 意味空間=水 / パラメータ=水の性質。
地形磁力モデルの別角度。
使いどころ: 意味空間の概念を初めて説明する場面。
（参照: `axis-log-02/integrated_knowledge.md`）

### 片栗粉モデル（Oobleck Model）
意味空間の操作を2段階で説明する比喩。

- Stage1（水＋熱 → 粘度変容）: 軸語彙による緩やかな地形操作 = 正常な距離操作
- Stage2（衝撃 → 硬化）: 過剰な制約語彙による地形崩壊 = Approach Failure

→ 操作の「成功モデル（Stage1）」と「失敗モデル（Stage2）」を両方内包。

海モデルとの整合:
粘度変容（ミクロ・局所操作）→ 海流（マクロ）に乗る → 意図到達。
衝撃硬化 = 巨大堤防 → 海流が止まる → 前提崩壊。

**[未整理]** 2段構造の完全統合設計は未完。
（参照: `axis-log-02/integrated_knowledge.md` §片栗粉2段構造）

### 海モデル（Ocean Model）
意味空間のマクロ構造比喩。海流 = 意味空間全体の方向性・流れ。
片栗粉モデル（ミクロ操作）と2層で対応する。

- 水槽（閉じた系）→ 海（開いた系）: スケールアップで系の性質が変わる
- 局所操作（軸語彙）が全体の海流に乗れば意図到達。断絶すると漂流。

**[未整理]** 先行例検索継続中。「世界初」表現は使わないこと。
（参照: `axis-log-02/integrated_knowledge.md` §スケールアップ=性質変化問題）

## chat-summary抽出・精査済み比喩

精査日: 2026-07-11

### おはじきモデル
ベクトル空間の初期説明用。意味のおはじきが散らばってて、丸で囲って抽出 = クラスタリング/距離操作の直感説明。
使いどころ: 意味空間を初めて説明する場面（第2章前半）。媒質三分割より具体的で直感的。

### 食紅メタファー
汚染の比喩。未知語・造語を意味空間に投入 → 周辺の確率密度が一瞬で歪む → Approach Failure（ハルシネーション）のトリガー。
使いどころ: 不可侵領域・Approach Failureの説明。「なぜ造語を使うと崩れるか」の直感説明。

### 信号分離比喩グループ（mp3 / カラオケボーカル消し / 電波フィルター）
同一概念の読者選択肢。内容同じ、表現違うだけ。読者に合わせて選ぶ。
AIが意味空間から関連成分だけ引き出す = 全部拾うんじゃなく特定周波数だけ抽出。
使いどころ: 距離操作・軸語彙による偏り誘導の説明。

---

### 折り紙テンソルモデル（ワープ補足説明）
ワープモデルの補足。「時空を歪める」がピンとこない読者向け。
紙を半分に折って両端を近づける = 意味空間を捻じ曲げて遠い概念を近づける。
使いどころ: ワープモデル説明直後の補足。読者がワープの直感を掴めていない場面。
